Questions

What does the server in the client server architecture do?

Servers are powerful computers which manages file servers and network traffic. The servers listens for requests from the client, and then sends a response back.

What does the client do?

Clients are computers which users use to run applications. In the client-server architecture a client uses the service provided by the server.

What is an IP address?

An Internet Protocol address is an address assigned to the devices connected to a TCP/IP-based network. It is used for unique identification and addressing hosts, servers and other endpoints.

What is DNS?

DNS is the Domain Name Server. It connects domain name to IP-address which makes it possible to send information to the correct location on the internet.

What is the difference between the Internet and WWW?

The internet is a networking infrastructure connecting endpoints. The World Wide Web is a way to access information through the internet.

Explain what HTML is

HTML, or HyperText Markup Language, is a computer language used for website creation. You can view HTML files through a browser - The browser translates the HTML text into a finished form, visible for the reader as the creator intended.

Why does inline style CSS override rules defined in style elements and external stylesheets?

The inline style has higher priority than internal and external stylesheets. This way you can change single elements that already has a style from other stylesheets.

Give a brief example of when to use ID ( #id ) and when to use classes ( .class ) in CSS.

IDs are for elements that are unique. This could be a single picture or a main title. Classes are for elements with the same style, like buttons.

What does RGBA mean and what colors can you express with it?

Red, green, blue, alpha. 24 bits RGB colors, and additionional 8 bit value for transparency.

Why do we include CSS files inside the head element and not inside the body element?

To try and keep the CSS off the critical path by downloading it as quick as possible. This way the page loads faster(Speeds up rendering).

What CSS selector matches all the p elements inside the article element in the following HTML?
